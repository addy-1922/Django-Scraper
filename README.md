🔗 Link Collector Scraper (Django)

A simple yet powerful link collector scraper built using Django. This project allows you to scrape, store, and manage links from various websites in a structured and searchable way.

🚀 Features
🌐 Scrape links from any given URL
💾 Store links in a database
🔍 Search and filter collected links
🧹 Remove duplicate entries automatically
📊 Admin panel for managing scraped data
⚡ Lightweight and extensible Django backend
🛠️ Tech Stack
Backend: Django (Python)
Database: SQLite (default, can be switched to PostgreSQL/MySQL)
Scraping: BeautifulSoup / Requests
Frontend: Django Templates (can be extended to React/Vue)
